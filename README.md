# 🏥 MediFlow AI

### Intelligent Hospital Management & Patient Prioritization Platform

## 📌 Overview

MediFlow AI is a modern Hospital Intelligence Platform designed to improve hospital operations through intelligent patient management, emergency prioritization, analytics, and resource optimization.

The platform enables hospitals to manage inpatient and outpatient records, prioritize emergency cases using a clinical rule-based decision engine, monitor hospital performance through interactive dashboards, and analyze patient flow in real time.

Built entirely using HTML, CSS, and JavaScript, MediFlow AI runs directly in the browser without requiring any backend server or database installation, making it lightweight, portable, and easy to deploy.

# ⚙️ Clinical Priority Scoring

The application uses a rule-based clinical decision engine that evaluates patient urgency based on multiple healthcare parameters.

### Parameters Considered

* Blood Pressure
* Heart Rate
* Oxygen Saturation
* Temperature
* Respiratory Rate
* Blood Sugar
* Hemoglobin
* White Blood Cell Count
* Serum Creatinine
* Age Risk
* Waiting Time
* Existing Comorbidities
* Chief Complaint

The calculated score categorizes patients into four urgency levels to support efficient emergency department triage.
